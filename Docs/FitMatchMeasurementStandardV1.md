# 핏매치 직접 측정 표준 v1

표준 식별자는 `fitmatch_standard_v1`이다. 이 표준은 사용자가 실제 옷을 줄자로 직접 측정할 때만 사용한다. 쇼핑몰에서 가져오거나 사이즈표에서 옮긴 값에는 적용하지 않는다.

## 공통 원칙

- 옷은 단추와 지퍼를 잠그고 평평한 바닥에 자연스럽게 편다.
- 원단을 당기거나 주름을 강제로 펴지 않는다.
- 단면 항목은 둘레로 환산하지 않고 한쪽 면의 폭을 cm로 기록한다.
- 시작점과 끝점이 정의와 다른 값은 비워 둔다.
- 입력 범위는 성인 의류의 단위·소수점 입력 실수를 차단하기 위한 안전 범위이며 사이즈 추천 기준이 아니다.

## 항목 정의

| 항목 | 측정 정의 | 입력 안전 범위 |
| --- | --- | ---: |
| 어깨너비 | 양쪽 어깨 봉제선의 가장 바깥점 사이 | 10~90cm |
| 가슴단면 | 겨드랑이 바로 아래 양쪽 끝 사이 | 15~100cm |
| 총장 | 앞면의 가장 높은 어깨점부터 밑단까지 | 15~180cm |
| 소매길이 | 어깨 봉제선부터 소매 끝까지 | 3~110cm |
| 허리단면 | 허리단 양쪽 끝 사이 | 15~90cm |
| 엉덩이단면 | 엉덩이 부분의 가장 넓은 지점 양쪽 끝 사이 | 20~100cm |
| 허벅지단면 | 가랑이 봉제점 바로 아래부터 바깥쪽 끝까지 | 10~60cm |
| 밑위 | 앞면 가랑이 봉제점부터 허리단 위까지 | 10~50cm |
| 밑단단면 | 한쪽 바짓단의 양쪽 끝 사이 | 5~90cm |
| 발길이 | 신발 안쪽 뒤꿈치 끝부터 발가락 끝까지 | 10~40cm |
| 밑가슴단면 | 밑가슴 밴드 양쪽 끝 사이 | 15~80cm |

## 버전 관리

측정 위치나 정의가 바뀌면 기존 표준을 수정하지 않고 새 버전을 만든다. 저장된 각 직접 측정 레코드는 사용한 표준 버전을 보존한다.
