# FitMatch Product Vision

작성일: 2026-07-07

## 제품 정의

FitMatch는 단순 사이즈 계산기가 아니라, **내 옷을 기준으로 다른 옷의 핏을 비교하는 서비스**다.

사용자는 이미 입어본 옷, 핏이 마음에 드는 옷, 자주 입는 옷을 기준 옷으로 저장한다. 이후 새로 구매하려는 상품 URL을 입력하면 FitMatch는 해당 상품의 실측 사이즈를 불러오고, 내 옷장 데이터와 비교해 가장 비슷한 핏의 사이즈를 추천한다.

## 핵심 Pain Point

온라인 쇼핑에서는 같은 `L`, `XL` 사이즈라도 브랜드, 상품, 카테고리, 핏 설계에 따라 실제 착용감이 크게 다르다.

대표적인 문제는 다음과 같다.

- 같은 브랜드 안에서도 상품별 실측이 다르다.
- 같은 `L` 사이즈라도 무신사, 유니클로, 29CM 입점 브랜드마다 핏이 다르다.
- 쇼핑몰 사이즈표를 직접 비교하기 번거롭다.
- 사용자는 본인에게 잘 맞는 옷의 실측값을 기억하지 못한다.
- 리뷰 기반 추천은 개인 옷장 기준이 아니기 때문에 정확도가 낮다.

## 핵심 해결

FitMatch는 사용자가 이미 잘 맞는 옷을 기준 옷으로 저장하게 한다.

이후 새 상품 URL을 넣으면 다음 과정을 통해 추천한다.

1. 상품 URL 분석
2. 상품명, 브랜드, 출처, 카테고리, 세부 카테고리 추출
3. 사이즈별 실측값 추출
4. 내 옷장 안에서 가장 적합한 기준 옷 선택
5. 실측 차이 계산
6. 추천 사이즈와 Fit Confidence 표시
7. 추천 이유와 비교 기준 표시

핵심 문장은 다음과 같다.

> 내 옷장에 있는 기준 옷과 가장 비슷한 사이즈를 찾아준다.

## 현재 전략

v0.1의 최우선 목표는 **무신사 상품 URL을 안정적으로 지원하는 것**이다.

현재 전략은 다음 순서로 확장한다.

1. 무신사 완성
2. 유니클로 공식몰 지원
3. 29CM 지원
4. W컨셉 지원
5. 쿠팡 지원

한 번에 모든 쇼핑몰을 지원하지 않는다. 무신사에서 상품 URL 분석, 메타데이터 추출, actual-size API 연동, 추천 결과 저장까지 완성한 뒤 같은 구조를 다른 쇼핑몰에 확장한다.

## 핵심 기능

### 내 옷장

사용자가 보유한 옷 또는 핏이 마음에 드는 옷을 저장한다.

저장 정보:

- 상품 출처
- 브랜드
- 성별
- 카테고리
- 세부 카테고리
- 상품명
- 사이즈
- 실측값
- 핏
- 만족도
- 대표 옷 여부

### URL 기반 상품 비교

사용자가 쇼핑몰 상품 URL을 입력하면 상품 정보를 자동으로 분석한다.

v0.1 기준 핵심 대상은 무신사 URL이다.

### 추천 사이즈

상품의 사이즈별 실측값과 내 옷장 기준 옷의 실측값을 비교해 가장 차이가 작은 사이즈를 추천한다.

### Fit Confidence

추천 결과에는 단순 추천 사이즈뿐 아니라 추천 신뢰도를 함께 표시한다.

신뢰도는 다음 요소를 반영한다.

- 같은 플랫폼인지
- 같은 브랜드인지
- 같은 세부 카테고리인지
- 같은 대분류인지
- 대표 옷인지
- 유효 실측 항목 수가 충분한지
- 사용자 선택 임시 비교인지

### 추천 기록

비교 결과는 RecommendationHistory로 저장한다.

기록 화면에서 다음 정보를 다시 볼 수 있다.

- 상품명
- 브랜드
- 상품 출처
- 추천 사이즈
- 추천도
- 비교 기준 옷
- 쇼핑몰 이동
- 다시 비교

### 상품 이미지 기반 기록

상품 이미지 URL은 무신사 메타데이터에서 추출한다.

향후 기록 화면과 추천 화면에서 상품 이미지를 함께 보여주는 방향으로 확장한다.

## 장기 비전

FitMatch의 장기 목표는 개인별 **Fit DNA**를 만드는 것이다.

Fit DNA는 다음 데이터를 축적해 만든다.

- 사용자가 만족한 옷
- 사용자가 불편했던 옷
- 선호 핏
- 선호 브랜드
- 카테고리별 기준 실측
- 플랫폼별 사이즈 편차
- 브랜드별 핏 성향

장기적으로는 플랫폼/브랜드별 측정 호환성 DB를 만든다.

예:

- 무신사 반소매 티셔츠 L과 유니클로 반소매 티셔츠 XL의 실측 호환성
- 특정 브랜드의 오버핏 경향
- 특정 플랫폼의 사이즈표 표기 방식

최종적으로 FitMatch는 사용자가 어떤 쇼핑몰에서 어떤 옷을 보더라도, 내 옷장을 기준으로 구매 실패 확률을 줄여주는 개인 맞춤 핏 플랫폼이 된다.
